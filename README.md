
# AI-Powered E-Commerce Sales & Customer Intelligence Platform

In today's competitive e-commerce landscape, understanding customer behavior and sales performance is essential for making informed business decisions. This project was built to transform raw e-commerce data into meaningful insights that help businesses improve customer retention, optimize sales strategies, and drive growth.

Using SQL, Python, Excel, and Power BI, I analyzed customer transactions, product performance, and sales trends to uncover valuable business insights. The data was cleaned and processed using Python (Pandas and NumPy), followed by Exploratory Data Analysis (EDA) to identify patterns, customer preferences, and revenue opportunities.

To make the insights easy to understand, I developed interactive Power BI dashboards that provide a clear view of key metrics such as revenue, customer segments, top-performing products, and regional sales performance. These dashboards allow users to explore business performance and make data-driven decisions with confidence.

The project also incorporates Machine Learning to predict customer churn and forecast future sales. By identifying customers who are likely to stop purchasing and estimating future demand, businesses can take proactive steps to improve customer engagement and plan resources effectively.

Additionally, AI-generated business recommendations are used to convert analytical findings into actionable suggestions, helping stakeholders make smarter strategic decisions.

### Key Highlights

* Cleaned and analyzed e-commerce datasets using SQL and Python
* Performed Exploratory Data Analysis (EDA) to uncover trends and customer insights
* Built interactive Power BI dashboards for business reporting
* Developed machine learning models for churn prediction and sales forecasting
* Generated AI-powered recommendations to support business growth
